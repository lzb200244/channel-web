引言

> 该项目是用于2023年秋季的毕设。基于drf+vue3以及相关技术栈开发。
> 支持群聊、私聊、@GPT等功能，探索并实现基于websocket协议的即时通讯系统。
> 熟悉一些常见设计模式与架构思想等。

## 技术选型

### [ 前端](https://github.com/lzb200244/easychannel-back)

-   vue3+ts+eslint

-   ant-design-vue3

-   mysql

### 中间件

-   redis
-   rabbitmq

### 第三方

-   百度大模型（文心一言）
